Semi - Story Board

관리자 메인 페이지(로그인전)


로그인을 안했을 시 로그인 안내문구와 로그인 페이지로 이동가능하다.


관리자 로그인 페이지


로그인 성공시에만 메인페이지로 이동가능하며, 실패시 이동할 수 업다.


관리자 메인 페이지


로그인 성공시, 이동가능한 메뉴(회원관리, 문의, 신고, 로그아웃)들이 나타난다.


회원관리 페이지


1.회원관리 페이지로 가능

2.알림보내기 페이지로 이동

3.클릭시 alert로 회원탈퇴를 한번 더 확인 후 탈퇴시킨다.

4.회원을 아이디로 검색가능


회원관리 - 레시피 목록


1.회원이 등록한 레시피를 목록으로 보여주며 해당페이지로 이동 가능하다.

2.레시피를 목록에서 바로 삭제가능하며, alert로 삭제를 한번 더 확인한다.


회원관리 -레시피 상세페이지


1.레시피 페이지로 이동 후 관리자 로그인의 세션값을 받아오게 되면 글삭제 버튼이 생성되게 하였고 글삭제가 가능하다.

2.관리자 로그인 세션갑을 받아오게 되면 댓글 삭제 버튼이 생기고 삭제가 가능하다.


회원관리 - 알림보내기


회원에게 상황에 맞는 알림을 셀렉트박스로 선택하여 보낼 수 있다.


회원관리 - 알림 목록


관리자가 보낸 알림을 회원의 알림 목록에서 확인 가능하다.


문의관리 - 문의글 목록


1.문의글 목록 상세페이지로 이동 가능

2.아이디로 문의글 검색가능


문의관리 - 문의글 상세보기


문의를 한 회원의 정보와 내용을 확인가능


문의관리 - 문의답변


문의글의 내용 확인 후 답변글 작성 가능


신고관리 - 신고글 목록


1.신고글 목록 상세페이지로 이동 가능

2.아이디로 신고글 검색가능


신고관리 - 신고글 상세페이지


1.회원이 신고하려는 페이지로 이동하여 신고내역 확인가능

2.신고글 처리 후 알림 페이지로 이동

DB 설계

Final - Story Board

관리자 메인 페이지(로그인전)


1.슬라이드 배너를 넣어 광고 효과를 주었다.

2.최신 공지사항 글 5개를 보여준다.

3.최신 우수봉사자 글 1개를 보여준다.


우수봉사자 - 목록


1.그 달의 봉사시간이 가장 많았던 우수봉사자들의 인터뷰 목록을 조회 가능하다.

2.제목과 내용으로 글 검색 가능하다.

3.페이지네이션, 페이지당 10개의 게시글 보여준다.


우수봉사자 - 상세페이지


우수봉사자의 정보(이름, 총 봉사시간)를 확인 가능하다.


공지사항 - 목록


1.공지사항 목록을 조회 가능하다.

2.제목과 내용으로 글 검색 가능하다.

3.페이지네이션, 페이지당 10개의 게시글 보여준다.


봉사실적 조회


1.조회하고 싶은 기간을 선택해서 검색이 가능하다.

2.봉사 실적 내용을 확인가능하다.

3.봉사실적내용을 PDF파일로 다운로드가 가능하다.


봉사실적인증서 PDF 다운로드


봉사내용(장소,시간,기간,현재 다운로드 받는 날짜)을 확인 후 PDF파일로 다운로드 가능하다.
